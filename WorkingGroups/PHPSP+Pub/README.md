## Resumo

O PHPUBSP é o encontro mensal focado em conteúdo em um ambiente descontraído.

---
## Descritivo

Evento mensal em Pub dividido em

* Apresentação
* Palestra
* Bate-papo

O espaço no Pub é reservado para o evento.
